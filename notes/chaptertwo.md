# 智能体发展史

## 基于符号与逻辑的早期智能体

### 物理符号系统假说 (PSSH)

核心观点：智能的本质就是符号的计算与处理。

两个论断：

- 充分性：任何物理符号系统都能产生通用智能行为。

- 必要性：任何能展现通用智能的系统，本质上必然是物理符号系统。

例子：专家系统 MYCIN 通过规则推理诊断疾病，证明了符号系统可以在复杂领域展现智能。

### 专家系统

结构：知识库 + 推理机 + 用户界面。

推理方式：

- 正向链：从事实出发逐步推理。

- 反向链：从目标出发逐步寻找证据。

例子：

- MYCIN：诊断细菌感染，推荐抗生素。它引入了“置信因子”来处理不确定性。

- 现实类比：就像医生问诊时一步步追问病史，最后得出结论。

### SHRDLU

特点：能在“积木世界”中理解自然语言并执行动作。

能力：

- 语言理解（指令、指代消解）。

- 规划与行动（先移开障碍再完成任务）。

- 记忆与问答（能回答“为什么做某事”）。

例子：用户说“把蓝色积木放到红色积木上”，系统会先移开阻挡的绿色积木，再执行指令。

意义：首次展示了综合性智能体的雏形。

### 符号主义的挑战

- 知识获取瓶颈：难以规模化编码专家知识。

- 常识问题：系统缺乏人类常识（如“水是湿的”）。

- 框架问题：难以判断哪些状态未改变。

- 脆弱性：规则之外的情况会导致系统失效。

例子：Cyc 项目试图编码人类常识，但几十年仍未完全成功。

## 基于规则的聊天机器人 —— ELIZA

设计思想：通过关键词匹配和句式转换，营造“理解”的假象。

工作机制：

- 模式匹配 → 文本替换 → 代词转换 → 输出回应。

例子：

- 用户说：“我为我的男朋友感到难过。”

- ELIZA回应：“你为什么会为你的男朋友感到难过？”

局限性：

- 缺乏语义理解。

- 无上下文记忆。

- 规则扩展性差。

现象：产生了“ELIZA效应”，用户误以为它能理解自己。

## 马文·明斯基的心智社会

观点：智能不是单一完美系统，而是由大量简单智能体协作涌现。

例子：

- 搭积木塔：高层机构激活子机构（找积木、抓取、放置），最终完成复杂任务。

- 如果“抓取”智能体失效，整个系统仍能部分运行，但任务可能失败。

启发：为多智能体系统 (MAS) 提供理论基础。

- 去中心化控制。

- 涌现式计算（如蚁群算法）。

- 智能体的社会性（通信、协商、组织结构）。

## 学习范式的演进与现代智能体

### 联结主义

思想：模仿神经网络，知识以连接权重分布式存储。

例子：图像识别中的卷积神经网络，不靠规则，而是通过训练自动识别猫和狗。

### 强化学习

机制：智能体通过与环境交互，在试错中学习最优策略。

例子：

AlphaGo：通过自我对弈学习围棋策略。

超级马里奥智能体：通过奖励信号（过关+1，死亡-1）学习跳跃和避障。

### 预训练与微调

流程：先在海量数据上预训练，再用少量任务数据微调。

例子：GPT 模型先学语言规律，再微调用于客服、编程助手。

突破：解决了符号主义时代的“知识获取瓶颈”。

### LLM驱动的智能体

架构：感知 → 规划 → LLM推理 → 工具调用 → 行动 → 反馈 → 记忆更新。

例子：智能旅行助手：先查天气，再推荐景点，形成闭环。

## 小结

智能体发展经历了 符号主义 → 联结主义 → 强化学习 → 大语言模型 的演进。

每一代范式解决了上一代的局限，同时带来新的挑战。

现代智能体融合了多种思想，形成了神经-符号结合的架构。

例子串联：

- MYCIN → 专业诊断。

- SHRDLU → 积木世界交互。

- ELIZA → 早期聊天机器人。

- AlphaGo → 强化学习突破。

- GPT → 预训练与涌现能力。
